Web platform built by a team of 7 at the JPMorgan Chase Code for Good Columbus 2025 Hackathon, designed to help students explore CLEP courses and track progress, with an admin dashboard for institutions to manage learners and courses.

I built the Modern States admin front-end, handled admin-specific login logic, and set up the final presentation.
